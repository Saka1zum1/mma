When no map is open you see the home screen. The left column, **Your Maps**, lists every map with a header showing the total map and location counts. The right column holds a **What's New** panel and the link to the **Manual**.

## What's New

The right column shows a scrollable release history pulled from the changelog. Each version section lists its user-facing changes. Versions newer than your installed app are visually marked so you can see what an update would bring.

## The search / create bar

- **Search maps.** Type to filter the list by map name or label as you type. Press <Kbd>Esc</Kbd> to clear.
- **Sort.** Order maps by **Name**, **Last opened**, **Date created**, or **Location count**.
- **New folder.** Type a name, then press the folder button to create a folder.
- **New map.** Type a name, then press the **+** button to create a map. If the search box has text and nothing matches, pressing <Kbd>Enter</Kbd> creates a map with that name and opens it.

## Map rows

- **Open.** Click a map's name to open it in its own editor window. The window title shows the map name.
- **Drag handle.** Drag a map by its handle onto a folder to move it; drag it to the root to move it out.
- **Edit.** The pencil renames the map and edits its **labels** (free-text keywords shown on the row and searchable).
- **Delete.** The trash icon deletes the map after a confirmation. You can also delete a map from inside its editor via the map rename dialog.
- **Row fields.** Each row can show location count, last opened, and date created. Choose which appear in **Settings → Advanced → Map List**.

## Folders

Folders are collapsible and show their map and location totals. Deleting a folder moves its maps back to the root rather than deleting them.
