The Core tab includes:

- **Map generator.** Generates Street View locations from coverage inside regions you draw, with extensive quality filters (camera generation, date ranges, official vs unofficial, minimum spacing, and more). It can follow linked panoramas to spread coverage. Additional options include searching within panorama descriptions, filtering by number of links, setting a fixed output zoom level, and adjusting the generation speed. Settings changes apply mid-run. An optional search-coverage overlay visualizes where the generator has searched. The generator session persists across view switches, so you can leave and return without losing progress.
- **Vali.** Generates locations from pre-built coverage data using the external Vali tool, streaming progress as it runs.
- **Distribution.** A live bar chart of how your locations are spread across countries, with flags and counts. It resolves countries offline from coordinates, so it works with zero enrichment; a **Coordinates** / **Metadata** toggle switches between coordinate-based counts and your enriched `countryCode` field when one is present.
- **Disambiguate.** Ranks metadata fields by how strongly they separate your active selections, helping you find what distinguishes one group from another.
- **Gradient.** Colors locations by a field value using gradient buckets. Supports the same date/number partition modes as **Apply metadata as tags**. A scope selector lets you color only the current selection, with bucket boundaries computed from the subset.
- **Pivot Table.** Cross-tabulates selections against location metadata.
- **JSON editor.** Shows the active location as editable raw JSON inside the location editor, with tag names resolved for readability.

A couple of cards are marked **coming soon** and cannot yet be enabled.
