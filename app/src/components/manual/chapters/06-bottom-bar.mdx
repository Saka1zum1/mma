The bar along the bottom holds the core map actions and status.

<Img name="bottom-bar.png" caption="The bottom bar: location count, Commit with diff, undo/redo, Seen, History, Import, Export." />

- **Location count.** The total number of locations on the map.
- **Commit.** Saves the current working changes as a version. The button is disabled when there is nothing to commit. When there are pending changes, a diff appears beside it: **+added**, **-removed**, **±modified**.
- **Undo** / **Redo**. Step backward and forward through your edits (<Kbd>Ctrl</Kbd>+<Kbd>Z</Kbd> / <Kbd>Ctrl</Kbd>+<Kbd>Y</Kbd>).
- **Seen.** Opens your pano visit history. See <ChapterLink id="seen" />.
- **History.** Opens the commit history for this map. See <ChapterLink id="version-history" />.
- **Import file.** Brings locations into the current map. See <ChapterLink id="importing-into-map" />.
- **Export.** Writes the map (or selection) out. See <ChapterLink id="exporting-map" />.
