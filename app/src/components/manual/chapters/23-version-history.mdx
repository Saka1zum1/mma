The **History** button opens the commit list for the map. Commits form a chain, each recording the change from the previous one.

## The commit table

- **Columns.** Date, a short hash, the change diff (+added / -removed / ±modified) or message, and the resulting location count.
- **View changes.** Click a row with changes to enter a full-map diff view: added locations appear green, removed red, and modified amber, with a sidebar showing aggregate counts per category. Rows with no diff are not clickable.
- **Revert / Restore.** The newest commit shows a **Revert** button; older commits show **Restore**. Either creates a new commit that returns the map to that commit's state (with a confirm step).

If the map has no commits yet, the dialog prompts you to press Commit to create the first.
