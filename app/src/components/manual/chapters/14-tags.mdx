Tags are colored on/off labels you attach to locations. They are boolean: a location either carries a tag or it does not.

## Creating and assigning

1. Create a tag from the Tags section; it gets a name and a color.
2. Assign tags while editing a location, using the tag bar at the bottom of the editor (type to add, click suggestions, X to remove).
3. **Quicktags.** While editing a location, number keys <Kbd>1</Kbd> through <Kbd>9</Kbd> toggle the first nine visible tags by position. These keys are rebindable. Quicktags and autocomplete suggestions follow your chosen tag sort order (name, count, or manual).
4. **Per-map hotkeys.** You can assign a dedicated hotkey to any tag from its edit dialog. Pressing the key while a location is open applies or removes the tag. These bindings are stored per-map, not globally. See <ChapterLink id="controls-hotkeys" />.
5. Click a tag pill in the Tags section to select every location carrying it; shift-click to extend a range.

## Colors, order, and visibility

- **Color.** Each tag has a color used for its markers, set with a color picker.
- **Order & sort.** Drag tags to reorder, or sort by name or count.
- **Visibility.** Hide a tag to declutter the map. A tag automatically becomes hidden when no location carries it.
- **Suggestion limit.** Cap how many tags appear in autocomplete and quicktag lists in **Settings → Street View → Tags → Suggestions shown**.

## Tree grouping

With **tree** view enabled (**Settings → Street View → Tags**), tags whose names contain `/` are grouped hierarchically, e.g. `Europe/France` nests under `Europe`. Parents show partial-selection indicators and inherit color.

## Tag tools

- **Apply metadata as tags.** Pick a metadata field and MMA creates one tag per distinct value, assigning each location the tag for its value (command palette). For example, applying **countryCode** turns a map into per-country tags in one step. Date and month fields offer a grouping choice (year, year-month, exact day, month of year, or hour of day), with an optional **Location timezone** checkbox; number fields can be bucketed by a fixed width (e.g. altitude in 500m bands). The same partition options are available in the Gradient plugin.
- **Find and replace in tag names.** Bulk-rename across tags (command palette).
- **Download tag counts as CSV.** Export per-tag counts (command palette).
