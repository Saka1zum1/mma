Bulk operations apply an action to many locations at once. Open them from the command palette (**Bulk Operations** group) or the relevant tool. Each opens a runner where you choose a **scope** and watch progress.

## Scope

Every bulk operation runs against either **All locations** or the **Current selection**. If you have an active selection, scope defaults to it.

## The operations

- **Validate locations.** Checks coverage and classifies problems. See <ChapterLink id="validation" />.
- **Enrich metadata fields.** Fetches metadata for the chosen fields. See <ChapterLink id="enrichment" />.
- **Set metadata field value.** Writes a chosen value into a field across the scope. Besides custom metadata fields, the picker also offers the built-in camera fields **Heading**, **Pitch**, and **Zoom**, so you can set a fixed point of view across many locations at once. For numeric fields you can enter an expression instead of a constant, such as `mod(heading + 180, 360)`. Expressions can reference any field on the location and use functions like `mod`, `clamp`, `abs`, `round`, `min`, and `max`.
- **Pan headings along road.** Sets each location's heading to face along the road, choosing **forwards** or **backwards**.
- **Clear metadata fields.** Removes chosen fields from the scope.
- **Pin locations to pano ID.** Resolves and pins a specific panorama to each location (sets **Load as pano ID**).
