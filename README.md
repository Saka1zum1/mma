Contains releases for local MMA.

Currently **not stable**. Core features should work reliably but there are no
guarantees for other features.

If you have a non-trivial bug, please
[create an issue](https://github.com/ccmdi/mma/issues) with:

- Fair description of what happened
- Steps to reproduce (if possible)
- `%LOCALAPPDATA%\app.map-making.local\logs\mma.log` included
